# dbd:log

引数: $ARGUMENTS

## 動作

1. 現在時刻を取得
2. 今日のタスクファイルを読み込み
3. Doneセクションにタイムスタンプ付きで追記
4. 記録内容を確認表示

## 手順

1. 日付と時刻を取得:
   ```bash
   date +%Y-%m-%d
   date +%H:%M
   ```

2. 今日のタスクファイルを読み込み

3. ファイルがない場合:
   - 「今日のタスクファイルがありません。`/dbd:today` を先に実行してください。」と案内
   - 終了

4. 引数から活動内容を取得

5. 引数がない場合:
   - AskUserQuestionで「何をやりましたか？」と質問

6. Doneセクションにタイムスタンプ付きで追記:
   - `## Done` セクションを探す
   - 新しい行を追加: `- HH:MM <活動内容>`
   - 既存のエントリは保持

7. ファイルを保存

## 出力形式

```
HH:MM に記録:
- <活動内容>
```
